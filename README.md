# Smart-Calculator
<h1>Ain't no ordinary calculator!<h1/>
**Smart-Calculator** is a sympy gui made in python
  ## Specifications
  - Run on Windows, Mac, Linux
  - Uses tkinter for UI
  - It acts basically as a normal calculator and make use of sympy features like plotting and calculating integrals and derivatives..
  
  ## How to Install
  You just need python3 on your system and install requirements.txt and you should be up and running!
