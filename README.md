# Installation ðŸ’»
Clone the repo:
```sh
git clone https://github.com/PROuserR/Smart-Calculator
```

Create a new virtual environment:
```sh
python -m venv .venv
```

Activate the virtual environment:
```sh
. .venv/Scripts/activate
```

Install the requirements:
```sh
pip install -r requirements.txt
```
